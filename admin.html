Check-in Security

The printed QR code on your poster links to a landing page that issues a short-lived, single-use token on each scan. That token is passed silently to the check-in form — it never appears in any URL, so the link cannot be bookmarked, photographed from the QR, or shared to skip in-person attendance. The token expires in 2 minutes and can only be used once.

Require an open session before accepting check-ins
Off (default): The QR code works at any time — members can check in whenever they scan it, day or night. Ideal for groups that run casual drop-in sessions and don't need a hard start/stop window.
On: An admin must press Open Session below before the first member scans. Anyone who scans outside that window sees a "No active session" screen and cannot submit. Use this when you want attendance recorded only during a specific rehearsal window, or when you're running multiple sessions in a week and want to keep them separate.
